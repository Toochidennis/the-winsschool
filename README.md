# The Wins School Website Redesign

Static HTML/CSS/JS redesign for The Wins School.

## Run locally

```bash
npm install --no-audit --no-fund
npm run dev
```

## Build

```bash
npm run build
```

## Apache hosting

Upload the built files, including `.htaccess`, to the hosting root. The `.htaccess` file enables clean URLs such as `/about` instead of `/about.html`.
